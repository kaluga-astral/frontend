# API

## Params

### useMenu(ref)

```ref``` - optional. Ref, в который будет установлен anchorRef. Если ref не указан, то хук сам создат anchorRef

## Returned

```ts
type UseMenuResult = {
  /**
   * Флаг состояния меню (открыто-закрыто)
   */
  open: boolean;
  /**
   * Элемент, к которому привязано меню
   */
  anchorRef: MutableRefObject<null>;
  /**
   * Обработчик открытия меню
   */
  handleCloseMenu: () => void;
  /**
   * Обработчик закрытия меню
   */
  handleOpenMenu: () => void;
};
```

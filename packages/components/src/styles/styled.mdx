# styled

Функция для стилизации компонентов из пакета [@emotion/styled](https://emotion.sh/docs/styled).

[Style guide](https://track.astral.ru/soft/wiki/display/FG/React#React-%D0%A1%D1%82%D0%B8%D0%BB%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F%D0%BA%D0%BE%D0%BC%D0%BF%D0%BE%D0%BD%D0%B5%D0%BD%D1%82%D0%BE%D0%B2) по стилизации компонентов.

## Example

```ts
import { styled } from '@astral/ui';

const CardWrapper = styled.article`
  width: 200px;
  height: 200px;
  padding: ${({ theme }) => theme.spacing(4)};

  background-color: ${({ theme }) => theme.palette.background.default};
`;
```

## Использование темы
Через тему обязательно должны задаваться:
- отступы (```theme.spacing```)
- размеры шрифтов (```theme.typography```)
- цвета (```theme.palette```)

## Пропсы для стилей
Если вашему компоненту стилей нужны пропсы, которые не должны передаваться дальше, то есть 2 подхода:

### Ручная проверка с использованием shouldForwardProp

```ts
import { styled } from '@astral/ui';

const Element = styled('div', {
  shouldForwardProp: (prop) => prop !== 'isActive',
})<{ isActive: boolean }>`
  background-color: ${({ theme, isActive }) =>
    isActive ? theme.palette.primary.main : theme.palette.background.default};
`;
```

### Именование пропсов с префиксом `$`

```ts
import { styled } from '@astral/ui';

const Element = styled('div')<{ $isActive: boolean }>`
  background-color: ${({ theme, $isActive }) =>
    $isActive ? theme.palette.primary.main : theme.palette.background.default};
`;
```
